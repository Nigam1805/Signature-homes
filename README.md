# Signature Homes SA – Website

A modern React website for **Signature Homes SA**, custom home builders across South Australia & Western Australia.

## Tech stack

- React 19, Vite, TypeScript  
- TailwindCSS, shadcn-style UI (Radix)  
- Motion (Framer Motion), React Router DOM  

## Commands

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # output in dist/
npm run preview  # preview production build
```

## Content

Edit all site text and images in **`src/content/siteContent.ts`** (nav, hero, about, contact details, footer, etc.).

## Deploy

Build output is in **`dist`**. Deploy that folder to Netlify, Vercel, or any static host.
